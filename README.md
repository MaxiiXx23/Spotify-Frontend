### Passo-a-passo para rodar a aplicação:

1. Entre na pasta Spotify-Frontend;
2. use o comando: **_yarn_** ou **_npm install/i_** para instalar os pacotes;
3. use o comando:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
